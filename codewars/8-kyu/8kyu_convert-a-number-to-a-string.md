# Convert a Number to a String!

We need a function that can transform a number into a string.

What ways of achieving this do you know?

Example:

```
123 --> "123"
999 --> "999"
```

---

## Solution

``` javascript
numberToString=n=>n.toString()
```

## Solution 2

``` javascript
numberToString=n=>`${n}`
```
